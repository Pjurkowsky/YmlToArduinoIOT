## Requirements

- Java 21 or later 
- Gradle (if not using Gradle Wrapper)
- [Arduino-cli](https://github.com/arduino/arduino-cli)

## Build and Run
1. build  \
```gradle build``` or ```./gradlew build```
2. run \
   ```gradle run``` or ```./gradlew run```

or just use intellji
